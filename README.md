## ğŸš€ Quick Start

### Prerequisites
- n8n installed (self-hosted or n8n.cloud account)
- Gmail account or SendGrid account
- Google account (for Drive, Sheets, Calendar)
- Gemini API key (free tier)

### Installation

1. **Import to n8n**
   - Open n8n
   - Click "Import from File"
   - Select `workflow.json`

2. **Configure Credentials**
   - Gmail API credentials
   - Google Drive API credentials
   - Gemini API key
   - ClickUp/Asana API key (optional)

3. **Test the Workflow**
   - Use test mode with sample data
   - Verify all integrations work
